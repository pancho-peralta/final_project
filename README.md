# Dashboard Interactivo: Crecimiento del PIB (% anual) - APEC

## Descripción

Este proyecto consiste en la creación de un dashboard interactivo desarrollado con Streamlit para analizar el crecimiento del PIB (% anual) de los países pertenecientes al Foro de Cooperación Económica Asia-Pacífico (APEC).

El dashboard permite seleccionar países, visualizar tendencias históricas, comparar el crecimiento individual con el promedio del grupo APEC y analizar indicadores estadísticos.

## Archivos del proyecto

- limpieza.ipynb: Notebook con el proceso de análisis exploratorio y limpieza de datos.
- inicio.py: Código del dashboard desarrollado en Streamlit.
- pib_limpio.csv: Dataset utilizado para la visualización.
